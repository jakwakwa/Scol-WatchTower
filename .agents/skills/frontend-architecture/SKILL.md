# Frontend Architecture

This skill contains rules for designing robust React component architectures, effective prop structures, and patterns like compound components.

Please refer to the markdown files in this directory for detailed guidelines.

## Included Rules

- **[Avoid Boolean Prop Proliferation](architecture-avoid-boolean-props.md)**: prevents unmaintainable component variants
- **[Use Compound Components](architecture-compound-components.md)**: enables flexible composition without prop drilling
- **[Prefer Composing Children Over Render Props](patterns-children-over-render-props.md)**: cleaner composition, better readability
- **[Create Explicit Component Variants](patterns-explicit-variants.md)**: self-documenting code, no hidden conditionals
