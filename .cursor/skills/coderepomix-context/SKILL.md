# Coderepomix Context

`coderepomix.md` is a packed representation of the repository generated by Repomix. It provides a single-file snapshot of the codebase for AI consumption.

## File Format

1. **Header**: Purpose, usage guidelines, directory structure
2. **File entries**: Each entry has:
   - `## File: path/to/file` — header with file path
   - Code block with file contents (comments removed, empty lines removed)
   - `⋮----` — delimiter between abbreviated code blocks within a file

## Usage Rules

- **Read-only**: Treat as reference. Never edit `coderepomix.md`. All changes go to the original repository files.
- **File paths**: Use the path in `## File: path/to/file` to locate and edit the real file.
- **Stale content**: The packed file may lag behind the repo. For current content, read the original file with the Read tool.
- **Security**: Handle as if it contains sensitive data; same care as the original repo.

## When to Use

- User attaches `@coderepomix.md` or asks for codebase overview
- Exploring structure (directory tree, file layout)
- Getting a broad view before diving into specific files
- Code review or analysis across multiple areas

## When to Prefer Original Files

- Editing or implementing changes
- Verifying current implementation
- Reading full content (packed version may truncate or compress)
